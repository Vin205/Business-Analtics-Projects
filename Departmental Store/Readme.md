Approach:

Cleaned and transformed retail dataset (5,000+ transactions) by removing duplicates, handling missing values, and standardizing formats
Built relational data model connecting Orders, Products, and Calendar tables to enable time-based analysis
Created 8 DAX measures including Total Sales, Profit Margin, YoY Growth, and Previous Year Comparisons

Key Insights Delivered:

Identified top 3 products generating 45% of total revenue, recommended increased inventory allocation
