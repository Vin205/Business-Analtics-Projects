Financial Report Analysis – Departmental Store
 Project Overview

This project focuses on analyzing the financial performance of a departmental store using business analytics techniques. The objective is to derive meaningful insights from sales data, track key performance indicators (KPIs), and support data-driven decision-making.

🎯 Objectives
Analyze sales trends across months and years
Evaluate profitability and discount impact
Identify top-performing products
Highlight loss-making segments
Compare current year vs previous year performance

📂 Dataset
Sample retail dataset containing:
Orders
Sales
Profit
Discounts
Product categories and segments
Data imported into the analytics tool for processing

🧹 Data Preparation & Transformation
Removed duplicate records
Cleaned inconsistent and missing data
Standardized data formats
Prepared structured tables for analysis

🧩 Data Modeling
Created relational data model
Established relationships between tables (e.g., Orders, Products, Dates)
Enabled time-based analysis (monthly & yearly trends)

📊 DAX Calculations

Key measures created using DAX:

Total Sales
Total Orders
Total Profit
Profit Margin
Total Discount

📈 Time Intelligence
Year-over-Year (YoY) Sales Comparison
Previous Year vs Current Year analysis

📉 Report Development
🔑 KPIs
Total Sales
Total Profit
Profit Margin
Total Orders

📊 Visualizations
Monthly and yearly sales trends
Product-wise sales performance
Segment-wise profitability

🎯 Insights Features
Highlighted Top 3 Products based on sales
Conditional formatting to:
Emphasize high-performing products
Highlight negative profit values by segment

🛠️ Tools & Technologies
